***

# 🚀 Quick Start Guide (Windows)

This guide assumes you have **GCC (MinGW)** installed and configured in your system PATH.

## 🛠️ Compilation

Open **Command Prompt (cmd)** or **PowerShell** in the project root directory and run the following two commands:

### 1. Compile the Scanner
```cmd
gcc scanner.c -o scanner.exe
```

### 2. Compile the Parser
```cmd
gcc main.c parser/parser.c parser/parse_table.c parser/stack.c parser/token_map.c parser/tree.c -I parser -o myparser.exe
```

---

## 🏃 How to Run

Simply run the parser executable. It will automatically call the scanner, generate the token file, and produce the parse tree.

```cmd
myparser.exe
```

### ⚙️ Important Note on `main.c`
If you get an error saying `scanner.exe` cannot be found, check your `main.c` file. Ensure the `system()` call matches the Windows filename:

**Correct for Windows:**
```c
int result = system("scanner.exe > tokens.txt");
```

---

## 📂 Expected Output Flow
1. **`input.c`** $\rightarrow$ Read by `scanner.exe`
2. **`tokens.txt`** $\rightarrow$ Generated by `scanner.exe`
3. **`myparser.exe`** $\rightarrow$ Reads `tokens.txt` and prints the **Parse Tree** to the console.

## 🧹 Cleanup
To remove the compiled files, run:
```cmd
del scanner.exe myparser.exe tokens.txt
```
